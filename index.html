Hockey Club Manager

Technologies used for this project: HTML | CSS | Angular 2 | Firebase

The goal of this project was to make use of what I learned about Angular 2, Firebase, CSS, JavaScript, and TypeScript. It's a hockey team manager, and allows a "team" to take a look at all the members, and allows a "manager" to edit the team's database. It has full CRUD functionality, and a password protected back end. Each member "tile" is clickble and will route you dynamically to that member's information. Please have a look!
